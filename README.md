# marilami

日付カウントダウン Discord Bot

## 環境変数

### Secret

|        変数名        |                説明                |             例              |
| :------------------: | :--------------------------------: | :-------------------------: |
|       `TOKEN`        |       Discord Botのトークン        |              -              |
|      `GUILD_ID`      |         ギルド(サーバ)のID         |              -              |
|     `CHANNEL_ID`     |       使用するチャンネルのID       |              -              |
|        `NAME`        |           対象となる名前           |           `太郎`            |
|       `ACTION`       |           対象となる動詞           |         `卒業する`          |
| `ACTION_EUPHONIC_TE` | 対象となる動詞の`て`で終わる音便形 |         `卒業して`          |
|      `DOOMSDAY`      |      対象となるタイムスタンプ      | `2025-03-21T00:00:00+09:00` |
